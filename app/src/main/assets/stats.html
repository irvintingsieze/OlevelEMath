1. Identify different statistical terms

1)First quartile : The data which is at 25% of the cumulative frequency
2) Second quartile: The data at the middle of cumulative frequency aka the median.
3) Third Quartile : The data which is at 75% of the cumulative frequency
5) Lower Quartile: It refers to the range of data below (not including) the median.
6) Upper Quartile: It refers to the range of data above(not including) the median.
7) Interquartile range: It refers to the difference between the first and third quartile.
8) Range: It refers to the difference between the last data and the first data.
9) Median: The data which is in the middle of all the given data.
10) Mean: The average of all the data combined.
11) Min: The smallest value
12) Max: The largest value
13) Standard Deviation: It defines how much the data deviates from the median.

2.Cumulative frequency graph

A cumulative frequency graph can be used to display data with increasing cumulative frequency.



From the graph alone, we can deduce the following:

3. Box and Whiskers Diagram

The box and whiskers diagram can be used to visualise data in the different quartiles.


With the box and whiskers diagram, we can see that
Median (at 50%) : 53
First Quartile (25%): 42
Third Quartile (75%): 62
Interquartile Range (Q3-Q1): 62-42 = 20
Min: 10
Max: 80
Range: 70

3. Standard deviation

Standard Deviation = $√{{∑f(x-x↖{-}^2 )}/{∑f}} $

OR

Standard Deviation = $√{{∑fx^2}/{∑f}- x↖{-}^2}$


Where $∑$ is the summation, f is the function, $x↖{-}$ is the mean.
