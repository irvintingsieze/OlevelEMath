2. Find the gradient of a curve by drawing a tangent and solving graphical problems

Q1. A new vehicle on experiment is set to move from point A to B in a straight line, which it reaches after 3s. The speed, s (Km/h) and time (s) follows a function $s=-3^3+3$.
(a) Draw a table to display the speed s where 0≤t≤3
(b) Using the scale 1cm to represent 0.5s in the horizontal axis and 1cm to represent 5km/h in the vertical axis, draw the graph of s against t for 0≤t≤3
(c) Using the graph, calculate the acceleration at t=2s
(d) Using the graph, estimate the distance travelled when 2.5≤t≤3. Express your answer in metres

(a)

t(s) 0 1 2 3
s(Km/h) 0 2 10 30

(b)

(c) Acceleration is the rate at which velocity (speed) is changing. (Since the car is moving in a straight line, the velocity and speed is the same for this case
Therefore, finding acceleration of t=2 is finding the gradient at t=2.
To find the gradient at t=2, we draw a tangent. The space between the tangent and the curve should be equal before and after t=2s

Remember the gradient = = $\text"Difference in s value" / \text"Difference in t value"$
The difference in s value = 17-3 = 14Km/h
The difference in t value = 2.5-1.5=1s
However the time and speed unit is not uniform. We can change the unit of time to complement the speed.
1s = $1/60$min = $1/3600$h
The acceleration at t=2 is 14 x $1/3600$=50400Km/$h^2$

(d) Distance = speed X Time.
Therefore to find the distance travelled, we calculate the area under the graph at 2.5≤t≤3
The time difference = 0.5s = $1/7200$h
Area under the graph = Area of trapezium= ${19+30}/2$X $1/7200$h = 0.003402(4sf)Km = 3.40m (3sf)
Total distance traveled is 3.40m (3sf).
