1. Identifying Matrices

Matrices can be used to solve questions where data can be represented in a table form.
Eg Qsn1: 2 students John and Mary went to the bookstore to buy some stationary. John bought a pencil, 2erasers and 3 pens while Mary bought 4 pencils, 5 pens and 6 erasers.

The following data can be represented in a table below.

Table

Pencil Eraser Pen
John 1 2 3
Mary 4 6 5

The following table can also be represented in a matrix $(\table \1, \2, \3; \4, \6,\5 )$.

The numbers from going from left to right are called rows. The first row is 1,2,3.
The numbers going from up to down are called columns. The first column is 1,4.

The size of the matrix can be represented by (rows) x (columns). In this case, it is a 2 x 3 matrix.

A row matrix is a matrix with only 1 row. Eg. ( 1 , 2 , 3 )
A column matrix is a matrix with only 1 column. Eg. $(\table \1, \2, \3)$

2. Addition and subtraction of Matrices

Matrices can be added or subtracted ONLY if they are of the same size.
A 3 x 3 matrix can only be added by another 3 x 3 matrix and so on.

To add the matrices, we just have to add the numbers of the matrices in the same position.
Eg. $(\table \1, \2, \3; \4, \5,\6 )$ + $(\table \2, \3, \4; \5, \6,\7 )$ = $(\table \3, \5, \7; \9, \11,\13 )$

The same goes for subtraction. Eg. $(\table \10, \9, \8; \7, \6,\5 )$ - $(\table \1, \2, \3; \4, \5,\6 )$ = $(\table \9, \7, \5; \3, \1,\-1 )$

3. Multiplication of Matrices

Matrix can be multiplied in 2 ways, by a number of by another matrix.

To multiply a matrix by a number, one simply multiply the number to all of the values in the matrix.
Eg. If a = $(\table \1, \2, \3; \4, \5,\6 )$ , 3a = $(\table \3, \6, \9; \12, \15,\18 )$

To multiply matrices, the number of column of the 1st matrix MUST be equal to the number of rows in the 2nd matrix.
The resulting matrix will have the number of rows of the 1st matrix and the number of columns in the 2nd matrix
Eg. A 2 x 3 matrix can multiply a 3 x 2 matrix but not a 2 x 3 matrix.

To multiply matrices, one must find the add the product of all the rows with all columns. The 1st row must multiply all the column first before going on to the second row.
An example will be $(\table \1, \2, \3; \4, \5,\6 )$ x $(\table \1, \2; \3, \4; \5,\6 )$ = $(\table \22, \28; \49, \64)$

1 x 1 + 2 x 3 + 3 x 5 = 22
1 x 2 + 2 x 4 + 3 x 6 = 28
4 x 1 + 5 x 3 + 6 x 5 = 49
4 x 2 + 5 x 4 + 6 x 6 = 64
Thus the resulting matrix is $(\table \22, \28; \49, \64)$.
