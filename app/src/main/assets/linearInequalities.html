1. Solve problems with inequalities

Q1. Ben and Tom have 18 toys in total. Ben have more than twice the amount of toys Tom have. Find the minimum number of toys Ben have.

Step 1: List and simplify the inequalities

Let B and T be the number of toys Ben and Tom have respectively.
B+T=18
B>2T

Step 2: Use substitution to ensure that there is only 1 variable left.

T=18-B
B>2(18-B)

Step 3: Simplify

B>36-2B
3B>36
B>12

The minimum number of toys Ben have is 13. (It is 13 as B is a value greater than 12)

2. Solve simultaneous linear inequalities

Q2. List all values of x where 4x+2≥2x-1≥5x-4.

Step 1: Split the inequalities such that there is only 1 inequality sign.

4x+1≥2x-1
2x-1≥5x-4

Step 2: Simplify both sides

2x≥-2
x≥-1

-3x≥-3
3x≤3 (rmb if u divide or multiply the equation by a negative number, the inequality sign changes)
x≤1

Step 3: Draw an number line and draw both inequalities. (For ≤ and ≥ draw a fully shaded end while for > and < draw hollow circle.)
From the number line, we can see the intersection, which would be where x lies.
Therefore, x = -1, 0, 1
