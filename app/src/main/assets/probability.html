Prerequisite:
The probability of an event A is denoted as P(A).
The probability of event A is $\text"P(A)" = \text"Possible outcomes for event A" / \text"All possible outcomes"$

1. Possibility diagrams

A possibility diagram is drawn when the event involves 2 requirements to be fulfilled.

Eg Qsn 1 If someone rolled a 4 sided die twice, what are the chances it rolls 1?


Using the possibility diagram, we can tell its $7/16$ at a glance with the diagram.

The possibility diagram is also useful to ease calculation as shown below
Eg Qsn 2 If someone rolled Die A and B, what is the probability that the sum is 8?

2.Types of probability events

1) "Either or" exclusive events
When an event occurs where there are 2 or more different outcomes, we can add the probabilities
Eg. P(roll a die showing 1 or 6) = P (die show 1) + P(die show 6)

2) "Not event A"
When the question ask the probability of event A not occurring.
P(A') = 1- P(A)

3) "Given that" events.
For any qsn that asks Event A given that event B occurs, apply this formula

$\text"P(A given B)" = \text"P(A and B)" / \text"P(B)"$

Another way is using a probability tree diagram which will be discussed later.

4) Sequence of events
If event B occurs after event A, then the probability of B occurring is
P(A then B) = P(A) x P(B)

3. Probability tree diagram

A probability tree diagram can be drawn to display a sequence of possible events.

Eg. Tom have a bag of beans. In the bag, there is 1 blue bean, 2 red beans and 3 white beans. Tom picked 2 beads, 1 after another.
a) What is the probability that Tom picked a blue bean on the first draw and red bean on the next draw?
b) What is the probability that Tom picked a red bean on second draw given that he picked he picked a blue bean on the first draw?


With the probability tree diagram, we can see that
(a) P ( Blue bean then red bean) = $1/6 x 2/5 = 1/15$

(b) P ( Red bean given that he picked blue bean on 1st draw) = $2/5$
We can conclude that as we assume that he already picked the blue bean. Therefore we just take the probability of the 2nd event.

3. Probability and sets

Sets can also be represented as a probability.
P (A and B) = $P (A ∪ B) = P (A) + P (B)$

For a sequence of events,
P (A followed by B) = $P (A ∩ B) = P (A) x P(B)$
