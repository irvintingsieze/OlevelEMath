Prerequisite crucial Knowledge

sinθ = $\text"opp" / \text"hyp"$
cosθ = $\text"adj" / \text"hyp"$
tanθ = $\text"opp" / \text"adj"$
Note that this can ONLY be applied to right angled triangles.

In this topic, we are going to learn how to
1. Find the sine and cosine of obtuse angles

1. Sine and cosine of obtuse angle

Recap: An obtuse angle is an angle between 90° and 180°, exclusive.
sin A = sin (180°-A)
cos A = -cos (180°-A)

2. sine and cosine rule

The image below will help illustrate sine, cosine rule AND the area of triangle.
sine rule
In the triangle ABC,
$\text"a" / \text"sin A"$ = $\text"b" / \text"sin B"$ = $\text"c" / \text"sin C"$
The reverse is true as well.
$\text"sin A" / \text"a"$ = $\text"sin B" / \text"b"$ = $\text"sin C" / \text"c"$

cosine rule

In the triangle ABC,
$a^2=b^2+c^2-2bc cosA$
$b^2=a^2+c^2-2ac cosB$
$c^2=a^2+b^2-2ab cosC$

From the above, we can also derive the following.
cosA= ${b^2+c^2-a^2}/{2bc}$
cosB= ${a^2+c^2-b^2}/{2ac}$
cosC= ${a^2+b^2-c^2}/{2ab}$
